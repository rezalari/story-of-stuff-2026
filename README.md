# The Story of Stuff — 18 Years Later (2007 → 2026 Fact-Check)

An interactive infographic fact-checking every statistic in Annie Leonard's 2007
documentary *The Story of Stuff* against the most recent (2024–2026) data from
EPA, FAO, UN/UNEP, BLS, IEA, USGS, ILO and the US Census.

**Live page:** see the repository's GitHub Pages site.

- `index.html` — the interactive infographic (filter by Worse / Better / Mixed)
- `StoryOfStuff_FactCheck_2026.xlsx` — the full sourced dataset

Compiled June 2026.
